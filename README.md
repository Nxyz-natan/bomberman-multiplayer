# Bomberman

A Bomberman with an AI that hunts you

## Features

- **Grid based movement** - you move to a dedicated grid in the map making it feel like the original bomberman
- **Ai that Hunts you** - Ai that goes to your position and place bombs around you and moves away from bombs to not die

##  Controls

| Action | Key |
|--------|-----|
| Move Left | ← Arrow / A |
| Move Right | → Arrow / D |
| Move up | ↑  Arrow / W |
| Move down | ↓ Arrow / S |
| Bomb |  Space/Enter |
